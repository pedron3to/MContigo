# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/pedron3to/MContigo.git
```


```bash
$ cd ..
$ cd MContigo
# Install Dependencies
$ yarn
```

```bash
$ yarn start
```

Go to http://localhost:9045 to see the result.
